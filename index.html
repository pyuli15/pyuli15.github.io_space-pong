How To Play:

Use the WASD keys to move the paddle left and right. Save the astronaut by hitting him with the ball. How many hits and misses can you get?





Inspiration:

This simple game was inspired by the classic pong game. My fascination with space combined with my love of the classic prompted me to create Space Pong. Using basic JavaScript, I created various game states including the start menu and the game state. There is no end, as I wanted the player to be able to play for as long as they want!
